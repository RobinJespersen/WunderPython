Wunderground Python API wrapper

WunderPython is an API wrapper for the Wunderground API.

At the moment the following data features are supported:

  • alerts
  • almanac
  • astronomy
  • conditions
  • forecast
  • forecast10day
  • history
  • hourly
  • hourly10day
  • rawtide
  • satellite
  • tide
  • webcams

Not yet supported:

  • currenthurricane
  • geolookup
  • planner
  • yesterday

Layers are also not yet supported.

Installation

pip install --pre WunderPython

Requirements

  • A Wunderground API key. Get one here

Usage

Create an instance of the API wrapper

from wunderpython import wunderground as w
wg = w.Wunderground("your secret api key")

Query for a location 

results = wg.search('Munich')

Get information about a location

print wg['Munich, Germany'].ll

Available location attributes are: l, lat, ll, lon, name, type, tz, tzs, zmw Furthermore you can access all features via attributes:

alerts = wg['Munich, Germany'].alerts

Only the history feature is a bit different because it needs additional parameters

historical_day = wg['Munich, Germany'].history['2014.05.23']

will return the recordings for 23. May 2014. You can also use slices to get time ranges:

historical_days = wg['Munich, Germany'].history['2014.05.23':'2014.05.29']

Or get multiple individual days:

two_historical_days = wg['Munich, Germany'].history['2014.05.23', '2014.05.29']
